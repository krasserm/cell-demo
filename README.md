# cell-demo

A small pipeline that turns a topic word into an interactive 3D model in the browser.

**Pipeline:** topic text → Gemini Nano Banana 2 (concept image) → Tripo image-to-3D (GLB) → static HTML viewer.

Given a topic like `neuron`, it generates a concept image, converts it to a textured GLB, and renders the model next to the concept image in `viewer.html`.

## Tech stack

- [Claude Code](https://claude.com/claude-code) — coding agent that built this demo
- [Gemini Nano Banana 2](https://ai.google.dev/gemini-api/docs/image-generation) — text-to-image model for the concept render
- [Tripo](https://www.tripo3d.ai) — image-to-3D API that produces the textured GLB
- [`<model-viewer>`](https://modelviewer.dev) — web component for rendering the GLB in the browser

## Setup

Create a `.env` in the project root:

```
GEMINI_API_KEY=...
TRIPO_API_KEY=...
```

## Usage

**1. Generate the assets.** Writes `output/concept.png` and `output/model.glb`. `uv run` reads the PEP 723 inline metadata in `generate.py` and provisions dependencies automatically:

```
uv run generate.py "neuron"
```

**2. Serve the viewer:**

```
python3 -m http.server
```

Open <http://localhost:8000/viewer.html>.
